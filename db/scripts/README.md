# DB Scripts

Run all scripts from the `db` directory.

From the Rideshare root, `cd` into `db`.

## Bulk Load

```sh
sh scripts/bulk_load.sh
```

## pgbench

```sh
sh scripts/benchmark.sh
```

## List table comments

```sh
sh scripts/list_table_comments.sh
```

## Simulate bloat

```sh
sh scripts/simulate_bloat.sh
```
